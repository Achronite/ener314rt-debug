# ener314rt-debug
Test program to diagnose issues with ener314rt and node

## Check you have node installed, at least version 10
``$ node -v``

## To compile code
``$ node-gyp rebuild``
If you do not have node-gyp, please install using ``$ npm install -g node-gyp``

## To run
``$ node test.js``
